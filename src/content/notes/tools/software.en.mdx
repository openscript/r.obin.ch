Here I list the software I'm using.

## Web

There are many web-based tools I use, but here are some of the most important ones:

- [Crontab Guru](https://crontab.guru) for creating cron schedule expressions.
- [Regex101](https://regex101.com) for testing and developing regular expressions.
- [IT-Tools](https://it-tools.tech) a Swiss Army knife for IT professionals including converters and network tools.
- [Hoppscotch](https://hoppscotch.io) for API development.
- [DevDocs](https://devdocs.io) for offline access to documentation.
- [.gitginore.io](https://gitignore.io) for creating `.gitignore` files.

## Linux

I'm using [Arch Linux](https://archlinux.org) with:

- [LibreOffice](https://libreoffice.org) as office suite.
- [Visual Studio Code](https://code.visualstudio.com) as code editor.
- [Firefox](https://www.mozilla.org/en-US/firefox) as web browser.

## Android

I'm using [GrapheneOS](https://grapheneos.org) with:

- [Aurora Store](https://gitlab.com/AuroraOSS/AuroraStore) as Google Play Store client.
- [Neostore](https://github.com/NeoApplications/Neo-Store) as F-Droid client.
